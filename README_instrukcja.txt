
1. Wgraj pliki index.html, login.html i supabaseClient.js do swojego repozytorium GitHub Pages.
2. Domyślnie startuj ze strony login.html (np. podlinkuj ją jako stronę startową).
3. Użytkownik loguje się przez Google, następnie wpisuje hasło 'rutynadw'.
4. Wszystkie checkboxy zapisują się w Supabase – oddzielnie dla każdego użytkownika.
5. Reset ALL czyści wszystkie checkboxy także w bazie.
